To access this schedule you must be a subscriber.

You can view activity schedules under My Schedules.

You can view choice boards under My Choice Boards.

 

To create a schedule:

Press the New button on the tool bar.

Enter in the name of the schedule in the Right where it says name: Enter name here!

Enter the text for your schedule.

To add pictures to a schedule:

Select Pictures on the main toolbar

Highlight a category and left mouse click to view the pictures in the selected category.

Select a picture by moving the mouse over the desired picture and left mouse click. A black outline goes around the picture.

Select a radio button in the edit pane in the Right where you want the picture to go.

Press left-right arrow on main toolbar to move the picture into the schedule.

The picture appears in the selected edit pane of your schedule.

 

To do a Web Search do the following:

Select Pictures from the toolbar. Fill in a subject.

Click Web Search!

 

You must be a subscriber to save a schedule.

To view more example schedules go to the home page.

For more help select Help on the main toolbar.

 

To print this text with Internet Explorer: right mouse click and select print

To view the list of schedules: Select List from the toolbar's dropdown listbox and press Go!
