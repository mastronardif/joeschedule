What is www.JoeSchedule.com ?

JoeSchedule.com contains activity schedules, step by step visual guides to completing tasks independently, pictures and a program that allows you to create your own activity schedules and save them on the Internet.

Activity schedules are used to increase independence, teach new skills, increase time spent on leisure activities, decrease tantrums, and make transitions easier by making expectations clear and consistent for individuals with disabilities.

What is a choice board?

A choice board is a set of related activities that a user can chose from.


What is the difference between a Choice Board and a Schedule?

A schedule is a list of activities that need to be completed in sequence. A choice board is a list of activities that are similar in nature that the user can chose from.


Choice boards add variety and the ability to choose to schedules. Choice boards can be used alone or embedded in a schedule.
